Cài đặt Java Link to heading

Một số máy tính có thể đã cài sẵn Java.

Kiểm tra Java trên máy tính Windows Link to heading

Để kiểm tra xem bạn đã cài đặt Java trên PC Windows hay chưa, hãy tìm kiếm Java trên thanh bắt đầu hoặc nhập lệnh sau vào Dấu nhắc lệnh (cmd.exe):

C:\Users\Your Name>java -version

Nếu Java đã được cài đặt, bạn sẽ thấy thông báo tương tự như thế này (tùy theo phiên bản):

java version "22.0.0" 2024-08-21 LTS
Java(TM) SE Runtime Environment 22.9 (build 22.0.0+13-LTS)
Java HotSpot(TM) 64-Bit Server VM 22.9 (build 22.0.0+13-LTS, mixed mode)

Nếu máy tính của bạn chưa cài đặt Java, bạn có thể tải xuống miễn phí tại oracle.com.

Lưu ý: Trong hướng dẫn này, chúng ta sẽ viết mã Java trong trình soạn thảo văn bản. Tuy nhiên, có thể viết Java trong Môi trường phát triển tích hợp (IDE), chẳng hạn như IntelliJ IDEA, NetBeans hoặc Eclipse, đặc biệt hữu ích khi quản lý các tập tin Java lớn hơn.

Bắt đầu nhanh Java Link to heading

Trong Java, mọi ứng dụng đều bắt đầu bằng một tên lớp, và lớp đó phải khớp với tên tệp.

Tạo tệp Java đầu tiên Link to heading

Hãy tạo tệp Java đầu tiên, có tên là Main.java. Tệp này có thể được tạo trong bất kỳ trình soạn thảo văn bản nào (như Notepad).

Tệp này sẽ chứa thông báo “Hello World”, được viết bằng mã sau:

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}

Đừng lo lắng nếu bạn không hiểu đoạn mã trên - chúng ta sẽ thảo luận chi tiết ở các chương sau. Bây giờ, hãy tập trung vào cách chạy đoạn mã trên.

Lưu mã trong Notepad dưới dạng Main.java. Mở Command Prompt (cmd.exe), điều hướng đến thư mục bạn đã lưu tệp và nhập javac Main.java:

C:\Users\Your Name>javac Main.java

Điều này sẽ biên dịch mã của bạn. Nếu không có lỗi trong mã, dấu nhắc lệnh sẽ đưa bạn đến dòng tiếp theo. Bây giờ, hãy nhập java Main để chạy tệp:

C:\Users\Your Name>java Main

Đầu ra sẽ là:

Hello World

Xin chúc mừng! Bạn đã viết và thực thi chương trình Java đầu tiên của mình. Quay lại Chuyển sang bài học tiếp theo
