Cú Pháp Java Link to heading

Java là một ngôn ngữ lập trình mạnh mẽ, phổ biến với cú pháp rõ ràng và dễ hiểu. Trong bài học này, chúng ta sẽ tìm hiểu về các cú pháp cơ bản trong Java, bao gồm các yếu tố như biến, kiểu dữ liệu, câu lệnh điều kiện, vòng lặp và cách viết các phương thức trong Java.

1. Cài Đặt Cú Pháp Java Link to heading

1.1. Cấu Trúc Một Chương Trình Java Link to heading

Một chương trình Java cơ bản có cấu trúc như sau:

public class Main {
    public static void main(String[] args) {
        // Câu lệnh thực thi
    }
}
  • public class Main: Định nghĩa một lớp có tên là Main. Tên lớp phải trùng với tên tệp và bắt đầu bằng chữ cái hoa.
  • public static void main(String[] args): Đây là phương thức main() bắt buộc, nơi chương trình bắt đầu thực thi.

1.2. Biến và Kiểu Dữ Liệu Link to heading

Java là một ngôn ngữ kiểu tĩnh, nghĩa là bạn phải khai báo kiểu dữ liệu khi khai báo biến.

Ví dụ: Link to heading 

public class Main {
    public static void main(String[] args) {
        int age = 25;                // Biến kiểu int
        String name = "John";        // Biến kiểu String
        double price = 19.99;        // Biến kiểu double
        boolean isJavaFun = true;    // Biến kiểu boolean
    }
}
  • int: Kiểu số nguyên.
  • double: Kiểu số thực (số thập phân).
  • String: Kiểu chuỗi (dãy ký tự).
  • boolean: Kiểu giá trị logic (true hoặc false).

1.3. Câu Lệnh Điều Kiện Link to heading

Câu lệnh điều kiện if, else if, và else giúp kiểm tra các điều kiện và thực thi mã dựa trên kết quả kiểm tra.

Ví dụ: Link to heading 

public class Main {
    public static void main(String[] args) {
        int age = 18;

        if (age >= 18) {
            System.out.println("Bạn đã đủ tuổi trưởng thành.");
        } else {
            System.out.println("Bạn chưa đủ tuổi trưởng thành.");
        }
    }
}

1.4. Vòng Lặp Link to heading

1.4.1. Vòng Lặp for Link to heading

Vòng lặp for được sử dụng khi bạn biết số lần lặp trước.

public class Main {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
            System.out.println(i);  // In ra các giá trị từ 0 đến 4
        }
    }
}

1.4.2. Vòng Lặp while Link to heading

Vòng lặp while được sử dụng khi bạn không biết số lần lặp trước và muốn lặp cho đến khi một điều kiện nào đó đúng.

public class Main {
    public static void main(String[] args) {
        int i = 0;
        while (i < 5) {
            System.out.println(i);  // In ra các giá trị từ 0 đến 4
            i++;
        }
    }
}

1.5. Phương Thức Link to heading

Phương thức trong Java được sử dụng để nhóm mã lại với nhau, giúp tái sử dụng mã dễ dàng hơn.

Ví dụ: Link to heading 

public class Main {
    // Phương thức không trả về giá trị
    public static void sayHello() {
        System.out.println("Hello, world!");
    }

    public static void main(String[] args) {
        sayHello();  // Gọi phương thức sayHello
    }
}

Phương thức có giá trị trả về: Link to heading 

public class Main {
    // Phương thức có giá trị trả về
    public static int addNumbers(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        int sum = addNumbers(5, 7);  // Gọi phương thức và lưu kết quả
        System.out.println("Tổng là: " + sum);
    }
}

1.6. Mảng Link to heading

Mảng trong Java dùng để lưu trữ nhiều giá trị trong một biến duy nhất.

Ví dụ: Link to heading 

public class Main {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};

        // Duyệt qua mảng và in ra các phần tử
        for (int i = 0; i < numbers.length; i++) {
            System.out.println(numbers[i]);
        }
    }
}

2. Tóm Tắt Cú Pháp Java Link to heading

  • Lớp và Phương thức: Java yêu cầu tất cả mã phải nằm trong lớp, và phương thức main() là nơi bắt đầu chương trình.
  • Biến: Bạn phải khai báo kiểu dữ liệu khi khai báo biến, và mỗi biến có một kiểu dữ liệu nhất định.
  • Điều kiện: Dùng if, else if, và else để kiểm tra điều kiện.
  • Vòng lặp: Dùng for hoặc while để lặp qua một tập hợp giá trị hoặc khi một điều kiện nào đó đúng.
  • Phương thức: Các phương thức dùng để thực hiện các hành động và có thể trả về giá trị hoặc không.
  • Mảng: Là cách để lưu trữ nhiều giá trị cùng một kiểu dữ liệu trong một biến duy nhất.

Ví Dụ Tổng Hợp Link to heading

Dưới đây là một ví dụ tổng hợp các cú pháp cơ bản trong Java:

public class Main {
    public static void main(String[] args) {
        // Khai báo và sử dụng biến
        int number = 10;
        String name = "Alice";

        // Sử dụng câu lệnh điều kiện
        if (number > 5) {
            System.out.println(name + " có số lớn hơn 5.");
        } else {
            System.out.println(name + " có số nhỏ hơn hoặc bằng 5.");
        }

        // Sử dụng vòng lặp
        for (int i = 0; i < 5; i++) {
            System.out.println("Giá trị i: " + i);
        }

        // Gọi phương thức
        int result = addNumbers(5, 7);
        System.out.println("Kết quả phép cộng: " + result);
    }

    public static int addNumbers(int a, int b) {
        return a + b;
    }
}

Đầu ra: Link to heading 

Alice có số lớn hơn 5.
Giá trị i: 0
Giá trị i: 1
Giá trị i: 2
Giá trị i: 3
Giá trị i: 4
Kết quả phép cộng: 12

Cú pháp Java là nền tảng cơ bản cho việc lập trình trong Java. Bằng cách hiểu và áp dụng các cú pháp cơ bản này, bạn có thể xây dựng các ứng dụng Java đơn giản và phức tạp hơn.

### Giải Thích:

1. **Cấu trúc cơ bản**: Mỗi chương trình Java cần có lớp và phương thức `main()`. Phương thức `main()` là nơi chương trình bắt đầu.
2. **Biến và kiểu dữ liệu**: Cần khai báo kiểu dữ liệu cho mỗi biến. Java hỗ trợ nhiều kiểu dữ liệu khác nhau như `int`, `double`, `String`, `boolean`.
3. **Điều kiện**: Sử dụng `if`, `else if`, và `else` để kiểm tra các điều kiện.
4. **Vòng lặp**: Java hỗ trợ cả vòng lặp `for` và `while` để lặp qua các phần tử hoặc lặp khi một điều kiện đúng.
5. **Phương thức**: Các phương thức trong Java có thể không trả về giá trị hoặc trả về một giá trị cụ thể. Chúng giúp tái sử dụng mã hiệu quả.
6. **Mảng**: Mảng là cách để lưu trữ nhiều giá trị trong một biến duy nhất.
