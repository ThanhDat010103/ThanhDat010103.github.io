Cú Pháp Cơ Bản của JavaScript Link to heading

JavaScript là một ngôn ngữ lập trình phổ biến được sử dụng rộng rãi trong phát triển web để tạo ra các trang web động và tương tác. Trong bài học này, chúng ta sẽ tìm hiểu cú pháp cơ bản của JavaScript, bao gồm cách khai báo biến, các cấu trúc điều khiển, hàm, và các thao tác cơ bản khác.

1. Khai Báo Biến Link to heading

JavaScript sử dụng ba từ khóa chính để khai báo biến: var, let, và const.

1.1. var Link to heading

  • var được sử dụng trong các phiên bản JavaScript trước đây. Biến khai báo bằng var có phạm vi toàn cục hoặc phạm vi hàm.
var x = 10;
console.log(x);  // Output: 10

1.2. let Link to heading

  • let được sử dụng để khai báo biến có phạm vi block (khối mã), giúp tránh một số vấn đề trong việc sử dụng var.
let y = 20;
console.log(y);  // Output: 20

1.3. const Link to heading

  • const được sử dụng để khai báo hằng số (biến không thể thay đổi giá trị sau khi được gán).
const z = 30;
console.log(z);  // Output: 30
// z = 40;  // Lỗi: không thể thay đổi giá trị của hằng số

2. Kiểu Dữ Liệu Link to heading

JavaScript hỗ trợ các kiểu dữ liệu cơ bản sau:

  • Number: Số nguyên và số thực.
  • String: Chuỗi ký tự.
  • Boolean: Giá trị đúng (true) hoặc sai (false).
  • Null: Không có giá trị nào.
  • Undefined: Biến chưa được gán giá trị.
  • Object: Một tập hợp các thuộc tính và phương thức.
  • Array: Mảng.

Ví Dụ: Link to heading 

let age = 25;        // Number
let name = "John";   // String
let isActive = true; // Boolean
let person = null;   // Null
let data;            // Undefined
let arr = [1, 2, 3]; // Array

3. Cấu Trúc Điều Kiện Link to heading

3.1. Câu Lệnh if Link to heading

Câu lệnh if kiểm tra một điều kiện và thực thi khối mã nếu điều kiện đúng.

let age = 18;
if (age >= 18) {
  console.log("Bạn đủ tuổi trưởng thành");
} else {
  console.log("Bạn chưa đủ tuổi trưởng thành");
}

3.2. Câu Lệnh switch Link to heading

Câu lệnh switch được sử dụng để kiểm tra nhiều điều kiện khác nhau.

let fruit = "apple";
switch (fruit) {
  case "apple":
    console.log("Đây là quả táo");
    break;
  case "banana":
    console.log("Đây là quả chuối");
    break;
  default:
    console.log("Quả không xác định");
}

4. Vòng Lặp Link to heading

4.1. Vòng Lặp for Link to heading

Vòng lặp for được sử dụng khi bạn biết số lần lặp.

for (let i = 0; i < 5; i++) {
  console.log(i);
}
// Output: 0 1 2 3 4

4.2. Vòng Lặp while Link to heading

Vòng lặp while được sử dụng khi bạn không chắc chắn số lần lặp, và điều kiện lặp sẽ được kiểm tra trước khi thực hiện.

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
// Output: 0 1 2 3 4

5. Hàm Link to heading

Hàm trong JavaScript được sử dụng để nhóm các câu lệnh lại và thực thi khi cần thiết. Có hai cách để khai báo hàm: sử dụng function và khai báo hàm dưới dạng biểu thức hàm.

5.1. Khai Báo Hàm với function Link to heading 

function greet(name) {
  console.log("Hello, " + name);
}
greet("Alice"); // Output: Hello, Alice

5.2. Hàm dưới dạng Biểu Thức Hàm (Anonymous Function) Link to heading 

const sum = function(a, b) {
  return a + b;
};
console.log(sum(3, 5)); // Output: 8

5.3. Hàm Mũi Tên (Arrow Function) Link to heading

Arrow function là một cú pháp ngắn gọn để khai báo hàm.

const multiply = (a, b) => a * b;
console.log(multiply(4, 5)); // Output: 20

6. Mảng (Array) Link to heading

Mảng trong JavaScript là một danh sách có thể chứa nhiều giá trị.

6.1. Khai Báo Mảng Link to heading 

let fruits = ["apple", "banana", "cherry"];
console.log(fruits); // Output: ["apple", "banana", "cherry"]

6.2. Truy Cập và Thao Tác với Mảng Link to heading 

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // Output: apple

fruits.push("orange");  // Thêm phần tử vào cuối mảng
console.log(fruits);     // Output: ["apple", "banana", "cherry", "orange"]

7. Đối Tượng (Object) Link to heading

Đối tượng trong JavaScript là một tập hợp các cặp khóa-giá trị.

7.1. Khai Báo Đối Tượng Link to heading 

let person = {
  name: "John",
  age: 30,
  greet: function() {
    console.log("Hello, " + this.name);
  }
};

console.log(person.name); // Output: John
person.greet();           // Output: Hello, John

8. Xử Lý Ngoại Lệ (Error Handling) Link to heading

JavaScript sử dụng try...catch để xử lý các ngoại lệ.

Ví Dụ: Link to heading 

try {
  let result = 10 / 0;
  if (result === Infinity) throw "Chia cho 0 không hợp lệ!";
} catch (error) {
  console.log("Lỗi: " + error);
}
// Output: Lỗi: Chia cho 0 không hợp lệ!

9. Câu Lệnh return Link to heading

Câu lệnh return được sử dụng để trả về giá trị từ hàm.

function add(a, b) {
  return a + b;
}
let sum = add(3, 4);
console.log(sum); // Output: 7

10. Kết Luận Link to heading

JavaScript là một ngôn ngữ linh hoạt và mạnh mẽ, được sử dụng rộng rãi trong phát triển web. Với cú pháp đơn giản và dễ học, JavaScript cho phép bạn xây dựng các trang web động và ứng dụng web mạnh mẽ. Việc hiểu rõ các khái niệm cơ bản về cú pháp JavaScript như khai báo biến, cấu trúc điều kiện, vòng lặp, hàm, và mảng sẽ giúp bạn nắm vững cách lập trình trong JavaScript.
