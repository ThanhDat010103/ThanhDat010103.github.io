DOM và Manipulation Link to heading

DOM (Document Object Model) là một giao diện lập trình ứng dụng (API) đại diện cho cấu trúc tài liệu HTML hoặc XML dưới dạng một cây các đối tượng. JavaScript có thể được sử dụng để thao tác với DOM, cho phép bạn thay đổi nội dung, cấu trúc và kiểu dáng của một trang web một cách động.

1. DOM là gì? Link to heading

DOM là một mô hình cây, trong đó mỗi phần tử HTML, thuộc tính và văn bản đều là một đối tượng trong cây này. JavaScript có thể truy cập và thay đổi các đối tượng này.

  • Document: Tài liệu HTML của trang web.
  • Object: Các phần tử (elements), thuộc tính (attributes), và nội dung văn bản (text) của trang.

2. Truy Cập Các Phần Tử DOM Link to heading

Để thao tác với DOM, bạn cần phải tìm các phần tử trong tài liệu HTML. JavaScript cung cấp nhiều phương thức để truy cập các phần tử này.

2.1. Truy Cập Bằng ID Link to heading

Sử dụng document.getElementById() để lấy một phần tử theo id.

let element = document.getElementById("myElement");
console.log(element);  // Output: Phần tử với id "myElement"

2.2. Truy Cập Bằng Class Link to heading

Sử dụng document.getElementsByClassName() để lấy tất cả các phần tử có class cụ thể.

let elements = document.getElementsByClassName("myClass");
console.log(elements);  // Output: Danh sách các phần tử có class "myClass"

2.3. Truy Cập Bằng Tag Name Link to heading

Sử dụng document.getElementsByTagName() để lấy tất cả các phần tử có tag cụ thể.

let elements = document.getElementsByTagName("p");
console.log(elements);  // Output: Danh sách các phần tử <p>

2.4. Truy Cập Bằng Query Selector Link to heading

Sử dụng document.querySelector() để lấy phần tử đầu tiên phù hợp với CSS selector, hoặc document.querySelectorAll() để lấy tất cả các phần tử phù hợp.

let element = document.querySelector(".myClass");
console.log(element);  // Output: Phần tử đầu tiên có class "myClass"

let elements = document.querySelectorAll("p");
console.log(elements);  // Output: Danh sách tất cả các phần tử <p>

3. Manipulate (Thao Tác) Với DOM Link to heading

Sau khi truy cập được các phần tử DOM, bạn có thể thao tác với chúng, như thay đổi nội dung, thuộc tính, hoặc kiểu dáng.

3.1. Thay Đổi Nội Dung Link to heading

Sử dụng innerHTML hoặc textContent để thay đổi nội dung của phần tử.

let element = document.getElementById("myElement");
element.innerHTML = "Nội dung mới";  // Thay đổi nội dung HTML của phần tử
element.textContent = "Nội dung văn bản mới";  // Thay đổi chỉ nội dung văn bản

3.2. Thay Đổi Thuộc Tính Link to heading

Bạn có thể thay đổi thuộc tính của phần tử bằng cách sử dụng setAttribute() hoặc element.attribute.

let imgElement = document.getElementById("myImage");
imgElement.setAttribute("src", "new-image.jpg");  // Thay đổi thuộc tính src của phần tử <img>

let linkElement = document.querySelector("a");
linkElement.href = "https://new-link.com";  // Thay đổi thuộc tính href của thẻ <a>

3.3. Thêm và Xoá Phần Tử Link to heading

Để thêm hoặc xóa các phần tử con của một phần tử, bạn có thể sử dụng các phương thức như appendChild(), removeChild(), và insertBefore().

let parentElement = document.getElementById("parent");
let newElement = document.createElement("div");
newElement.textContent = "Phần tử mới";
parentElement.appendChild(newElement);  // Thêm phần tử mới vào cuối

parentElement.removeChild(newElement);  // Xoá phần tử mới

3.4. Thay Đổi Kiểu Dáng Link to heading

Bạn có thể thay đổi kiểu dáng của các phần tử thông qua thuộc tính style.

let element = document.getElementById("myElement");
element.style.color = "red";  // Thay đổi màu chữ thành đỏ
element.style.fontSize = "20px";  // Thay đổi kích thước chữ

3.5. Thêm và Xoá Lớp CSS Link to heading

Để thêm hoặc xoá lớp CSS của phần tử, bạn có thể sử dụng classList.

let element = document.getElementById("myElement");
element.classList.add("newClass");  // Thêm lớp mới vào phần tử
element.classList.remove("oldClass");  // Xoá lớp cũ khỏi phần tử

4. Xử Lý Sự Kiện (Event Handling) Link to heading

JavaScript cung cấp một cách để xử lý các sự kiện (events) như click, hover, keypress, v.v. Bạn có thể sử dụng các phương thức như addEventListener() để gán các hàm xử lý sự kiện vào các phần tử DOM.

4.1. Thêm Sự Kiện Click Link to heading 

let button = document.getElementById("myButton");
button.addEventListener("click", function() {
  alert("Button clicked!");
});

4.2. Thêm Sự Kiện Hover Link to heading 

let element = document.getElementById("myElement");
element.addEventListener("mouseenter", function() {
  element.style.backgroundColor = "yellow";
});
element.addEventListener("mouseleave", function() {
  element.style.backgroundColor = "";
});

5. DOM Traversing (Duyệt DOM) Link to heading

Bạn có thể di chuyển qua các phần tử DOM con, anh chị em hoặc cha của một phần tử bằng các thuộc tính như parentNode, childNodes, nextSibling, và previousSibling.

5.1. Truy Cập Phần Tử Cha Link to heading 

let childElement = document.getElementById("myElement");
let parentElement = childElement.parentNode;
console.log(parentElement);  // Output: Phần tử cha của "myElement"

5.2. Truy Cập Các Phần Tử Con Link to heading 

let parentElement = document.getElementById("parent");
let children = parentElement.childNodes;
console.log(children);  // Output: Danh sách các phần tử con của "parent"

6. Kết Luận Link to heading

DOM (Document Object Model) cung cấp một cách để JavaScript tương tác và thao tác với nội dung và cấu trúc của một trang web. Với DOM, bạn có thể dễ dàng truy cập, thay đổi, thêm hoặc xóa các phần tử trong trang web của mình. Kết hợp với các sự kiện, DOM giúp bạn tạo ra các ứng dụng web động và tương tác.

Bằng cách sử dụng các phương thức và thuộc tính của DOM, bạn có thể thay đổi nội dung, kiểu dáng, hoặc thậm chí thay đổi cấu trúc của trang web mà không cần phải tải lại trang, mang lại trải nghiệm người dùng mượt mà và nhanh chóng. Quay lại Chuyển sang bài học tiếp theo
