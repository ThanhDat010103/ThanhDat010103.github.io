Lập Trình Hướng Đối Tượng trong Java Link to heading

Lập trình hướng đối tượng (OOP) là một phong cách lập trình dựa trên các đối tượng và các lớp. Java là một ngôn ngữ lập trình hướng đối tượng, và bạn có thể sử dụng OOP để tổ chức và thiết kế các ứng dụng phức tạp.

Các Khái Niệm Cơ Bản trong Lập Trình Hướng Đối Tượng Link to heading

1. Lớp (Class) Link to heading

Lớp là một khuôn mẫu để tạo ra đối tượng. Nó định nghĩa các thuộc tính (biến) và phương thức (hàm) mà đối tượng của lớp đó có. Ví dụ, lớp Person có thể có các thuộc tính như tên, tuổi và phương thức như sayHello().

public class Person {
    // Thuộc tính
    String name;
    int age;

    // Phương thức
    public void sayHello() {
        System.out.println("Hello, my name is " + name);
    }
}

2. Đối Tượng (Object) Link to heading

Đối tượng là một thể hiện (instance) của lớp. Khi bạn tạo một đối tượng từ lớp, đối tượng này sẽ có các thuộc tính và phương thức mà lớp định nghĩa.

public class Main {
    public static void main(String[] args) {
        // Tạo đối tượng từ lớp Person
        Person person1 = new Person();
        person1.name = "John";
        person1.age = 25;
        
        // Gọi phương thức sayHello()
        person1.sayHello();
    }
}

3. Tính Kế Thừa (Inheritance) Link to heading

Kế thừa cho phép một lớp con (subclass) kế thừa các thuộc tính và phương thức của lớp cha (superclass). Điều này giúp tái sử dụng mã và mở rộng các lớp hiện có.

// Lớp cha
public class Animal {
    public void sound() {
        System.out.println("Animal makes a sound");
    }
}

// Lớp con
public class Dog extends Animal {
    @Override
    public void sound() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.sound(); // Gọi phương thức của lớp con
    }
}

4. Tính Đa Hình (Polymorphism) Link to heading

Đa hình cho phép một phương thức hoặc đối tượng có nhiều hình thức. Trong Java, đa hình có thể được thực hiện thông qua phương thức ghi đè (overriding) hoặc phương thức nạp chồng (overloading).

Ví dụ về ghi đè (Overriding):

public class Animal {
    public void sound() {
        System.out.println("Animal makes a sound");
    }
}

public class Cat extends Animal {
    @Override
    public void sound() {
        System.out.println("Cat meows");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myCat = new Cat(); // Đối tượng kiểu Animal nhưng là đối tượng của lớp Cat
        myCat.sound(); // Gọi phương thức của lớp Cat
    }
}

5. Tính Đóng Gói (Encapsulation) Link to heading

Đóng gói là việc ẩn các chi tiết cài đặt bên trong lớp và chỉ cung cấp những phương thức công khai để người khác sử dụng. Điều này giúp bảo vệ dữ liệu và ngăn ngừa việc truy cập trái phép.

public class Person {
    // Thuộc tính private (không thể truy cập trực tiếp từ bên ngoài)
    private String name;

    // Phương thức getter và setter để truy cập vào thuộc tính
    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

public class Main {
    public static void main(String[] args) {
        Person person1 = new Person();
        person1.setName("John");
        System.out.println(person1.getName()); // In ra tên của person1
    }
}

Tóm Tắt Link to heading

  • Lớp (Class): Định nghĩa các thuộc tính và phương thức của đối tượng.
  • Đối tượng (Object): Là thể hiện của lớp.
  • Kế thừa (Inheritance): Cho phép lớp con kế thừa thuộc tính và phương thức của lớp cha.
  • Đa hình (Polymorphism): Cho phép phương thức hoặc đối tượng có nhiều hình thức khác nhau.
  • Đóng gói (Encapsulation): Bảo vệ dữ liệu và chỉ cung cấp các phương thức công khai để truy cập.

Ví Dụ Tổng Hợp Link to heading 

public class Car {
    private String brand;
    private int speed;

    // Constructor
    public Car(String brand, int speed) {
        this.brand = brand;
        this.speed = speed;
    }

    // Getter và Setter
    public String getBrand() {
        return brand;
    }

    public void setBrand(String brand) {
        this.brand = brand;
    }

    public int getSpeed() {
        return speed;
    }

    public void setSpeed(int speed) {
        this.speed = speed;
    }

    // Phương thức
    public void drive() {
        System.out.println("The " + brand + " car is driving at " + speed + " km/h.");
    }
}

public class Main {
    public static void main(String[] args) {
        Car car = new Car("Toyota", 120);
        car.drive();
    }
}

Đầu ra: Link to heading 

The Toyota car is driving at 120 km/h.
