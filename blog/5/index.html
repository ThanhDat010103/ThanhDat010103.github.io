Java Collections Framework Link to heading

Java Collections Framework là một tập hợp các lớp và giao diện (interfaces) cho phép bạn lưu trữ, truy xuất, và thao tác với dữ liệu trong các cấu trúc dữ liệu khác nhau, như danh sách, tập hợp, và bản đồ. Framework này cung cấp một cách tiếp cận thống nhất để làm việc với các tập hợp dữ liệu trong Java.

Các Thành Phần Chính của Collections Framework Link to heading

  1. Interfaces (Giao Diện):

    • Các giao diện trong Collections Framework xác định các hành vi cơ bản mà các lớp phải tuân thủ.
    • Các giao diện quan trọng bao gồm:
      • Collection: Giao diện cơ sở cho các tập hợp.
      • List: Kế thừa từ Collection, đại diện cho một danh sách các phần tử có thể có thứ tự.
      • Set: Kế thừa từ Collection, đại diện cho một tập hợp các phần tử không có thứ tự và không có phần tử trùng lặp.
      • Queue: Kế thừa từ Collection, đại diện cho các hàng đợi (queues).
      • Map: Giao diện không phải là con của Collection, đại diện cho các cấu trúc dữ liệu khóa-giá trị (key-value pairs).

  2. Classes (Lớp):

    • Các lớp cụ thể triển khai các giao diện và cung cấp các tính năng thực tế cho các tập hợp dữ liệu.
    • Ví dụ:
      • ArrayList, LinkedList: Các lớp triển khai List.
      • HashSet, TreeSet: Các lớp triển khai Set.
      • HashMap, TreeMap: Các lớp triển khai Map.
      • PriorityQueue: Lớp triển khai Queue.

  3. Algorithms (Thuật Toán):

    • Các thuật toán chuẩn để sắp xếp, tìm kiếm và thao tác với các tập hợp dữ liệu, ví dụ: Collections.sort(), Collections.reverse().

  4. Iterator:

    • Cung cấp một cách tiếp cận thống nhất để duyệt qua các phần tử trong một Collection.

Các Loại Collection Link to heading

1. List Link to heading

List là một giao diện con của Collection cho phép bạn lưu trữ các phần tử theo thứ tự và có thể truy cập qua chỉ mục. Các phần tử trong List có thể trùng lặp.

Ví dụ về List:

import java.util.ArrayList;
import java.util.List;

public class ListExample {
    public static void main(String[] args) {
        List<String> fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");

        // In danh sách
        System.out.println(fruits);
    }
}

Kết quả:

[Apple, Banana, Orange]

2. Set Link to heading

Set là một giao diện con của Collection cho phép lưu trữ các phần tử không trùng lặp và không có thứ tự.

Ví dụ về Set:

import java.util.HashSet;
import java.util.Set;

public class SetExample {
    public static void main(String[] args) {
        Set<String> animals = new HashSet<>();
        animals.add("Cat");
        animals.add("Dog");
        animals.add("Cat");  // Phần tử trùng lặp sẽ bị bỏ qua

        // In tập hợp
        System.out.println(animals);
    }
}

Kết quả:

[Dog, Cat]

3. Queue Link to heading

Queue là một giao diện con của Collection, đại diện cho một hàng đợi FIFO (First In, First Out). Các phần tử được thêm vào cuối hàng đợi và loại bỏ từ đầu hàng đợi.

Ví dụ về Queue:

import java.util.LinkedList;
import java.util.Queue;

public class QueueExample {
    public static void main(String[] args) {
        Queue<String> queue = new LinkedList<>();
        queue.add("First");
        queue.add("Second");
        queue.add("Third");

        // In hàng đợi
        System.out.println(queue);

        // Loại bỏ phần tử đầu tiên
        queue.poll();
        System.out.println(queue);
    }
}

Kết quả:

[First, Second, Third]
[Second, Third]

4. Map Link to heading

Map là một giao diện không phải là con của Collection nhưng rất quan trọng trong Java Collections Framework. Nó lưu trữ các cặp khóa-giá trị, trong đó mỗi khóa là duy nhất.

Ví dụ về Map:

import java.util.HashMap;
import java.util.Map;

public class MapExample {
    public static void main(String[] args) {
        Map<String, Integer> studentGrades = new HashMap<>();
        studentGrades.put("John", 90);
        studentGrades.put("Jane", 85);
        studentGrades.put("Tom", 80);

        // In bản đồ
        System.out.println(studentGrades);

        // Truy xuất giá trị theo khóa
        System.out.println("John's grade: " + studentGrades.get("John"));
    }
}

Kết quả:

{John=90, Jane=85, Tom=80}
John's grade: 90

Thuật Toán và Công Cụ Khác Link to heading

Java Collections Framework cũng cung cấp các thuật toán chuẩn để thao tác với dữ liệu:

  • Collections.sort(): Sắp xếp một danh sách.
  • Collections.reverse(): Đảo ngược một danh sách.
  • Collections.shuffle(): Xáo trộn một danh sách.

Ví dụ về sắp xếp:

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class SortExample {
    public static void main(String[] args) {
        List<Integer> numbers = new ArrayList<>();
        numbers.add(5);
        numbers.add(2);
        numbers.add(8);
        numbers.add(1);

        // Sắp xếp danh sách
        Collections.sort(numbers);
        System.out.println(numbers);
    }
}

Kết quả:

[1, 2, 5, 8]

Kết luận Link to heading

Java Collections Framework cung cấp một tập hợp các công cụ mạnh mẽ giúp bạn làm việc với các cấu trúc dữ liệu như danh sách, tập hợp, hàng đợi và bản đồ.
