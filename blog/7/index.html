Java Input/Output (I/O) Link to heading

Java cung cấp một bộ API mạnh mẽ để xử lý các thao tác Input/Output (I/O). I/O trong Java bao gồm các thao tác đọc và ghi dữ liệu từ các nguồn như tệp tin, bộ nhớ, và mạng. Java cung cấp các lớp và giao diện để làm việc với I/O dưới hai hình thức chính: byte-oriented I/O và character-oriented I/O.

1. I/O trong Java Link to heading

1.1. Byte-oriented I/O Link to heading

Byte-oriented I/O làm việc với dữ liệu ở dạng byte. Nó phù hợp để xử lý các tệp nhị phân, như hình ảnh, video, và âm thanh. Các lớp trong java.io như FileInputStream và FileOutputStream dùng để đọc và ghi dữ liệu byte.

Ví dụ về đọc và ghi byte: Link to heading 

import java.io.FileInputStream;
import java.io.FileOutputStream;
import java.io.IOException;

public class ByteIOExample {
    public static void main(String[] args) {
        try {
            // Ghi dữ liệu vào tệp
            FileOutputStream fos = new FileOutputStream("example.txt");
            String data = "Hello, Java I/O!";
            fos.write(data.getBytes());
            fos.close();

            // Đọc dữ liệu từ tệp
            FileInputStream fis = new FileInputStream("example.txt");
            int content;
            while ((content = fis.read()) != -1) {
                System.out.print((char) content);
            }
            fis.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Kết quả:

Hello, Java I/O!

1.2. Character-oriented I/O Link to heading

Character-oriented I/O làm việc với dữ liệu ở dạng ký tự. Điều này thích hợp với việc xử lý các tệp văn bản. Các lớp như FileReader và FileWriter hỗ trợ việc đọc và ghi các ký tự.

Ví dụ về đọc và ghi ký tự: Link to heading 

import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;

public class CharacterIOExample {
    public static void main(String[] args) {
        try {
            // Ghi dữ liệu vào tệp
            FileWriter writer = new FileWriter("example.txt");
            String data = "Chào bạn, Java I/O!";
            writer.write(data);
            writer.close();

            // Đọc dữ liệu từ tệp
            FileReader reader = new FileReader("example.txt");
            int content;
            while ((content = reader.read()) != -1) {
                System.out.print((char) content);
            }
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Kết quả:

Chào bạn, Java I/O!

2. Các Lớp Cơ Bản trong Java I/O Link to heading

Java cung cấp nhiều lớp để xử lý I/O. Dưới đây là các lớp phổ biến:

2.1. FileInputStream và FileOutputStream Link to heading

  • FileInputStream: Dùng để đọc byte từ tệp.
  • FileOutputStream: Dùng để ghi byte vào tệp.

2.2. FileReader và FileWriter Link to heading

  • FileReader: Dùng để đọc ký tự từ tệp văn bản.
  • FileWriter: Dùng để ghi ký tự vào tệp văn bản.

2.3. BufferedReader và BufferedWriter Link to heading

Các lớp này được sử dụng để đọc và ghi dữ liệu nhanh chóng hơn, sử dụng bộ đệm (buffer).

  • BufferedReader: Đọc dữ liệu một cách nhanh chóng từ tệp văn bản.
  • BufferedWriter: Ghi dữ liệu một cách hiệu quả vào tệp văn bản.

Ví dụ về sử dụng BufferedReader và BufferedWriter: Link to heading 

import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;

public class BufferedIOExample {
    public static void main(String[] args) {
        try {
            // Ghi dữ liệu vào tệp bằng BufferedWriter
            BufferedWriter writer = new BufferedWriter(new FileWriter("example.txt"));
            writer.write("Dữ liệu được ghi bằng BufferedWriter!");
            writer.close();

            // Đọc dữ liệu từ tệp bằng BufferedReader
            BufferedReader reader = new BufferedReader(new FileReader("example.txt"));
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Kết quả:

Dữ liệu được ghi bằng BufferedWriter!

2.4. PrintWriter Link to heading

PrintWriter là một lớp rất hữu ích để ghi văn bản vào tệp hoặc ra màn hình, hỗ trợ các phương thức tiện dụng như println() và printf().

Ví dụ về sử dụng PrintWriter: Link to heading 

import java.io.PrintWriter;
import java.io.IOException;

public class PrintWriterExample {
    public static void main(String[] args) {
        try {
            PrintWriter writer = new PrintWriter("example.txt");
            writer.println("Hello, PrintWriter!");
            writer.printf("Số nguyên: %d, Số thực: %.2f", 100, 3.14);
            writer.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Kết quả (trong tệp example.txt):

Hello, PrintWriter!
Số nguyên: 100, Số thực: 3.14

3. Xử Lý Ngoại Lệ trong I/O Link to heading

Khi làm việc với I/O, bạn sẽ thường xuyên gặp phải các ngoại lệ như IOException. Để xử lý các ngoại lệ này, Java cung cấp cơ chế try-catch-finally.

Ví dụ về xử lý ngoại lệ I/O: Link to heading 

import java.io.FileReader;
import java.io.IOException;

public class IOExceptionExample {
    public static void main(String[] args) {
        try {
            FileReader reader = new FileReader("nonexistentfile.txt");
            reader.read();
        } catch (IOException e) {
            System.out.println("Lỗi: " + e.getMessage());
        } finally {
            System.out.println("Đảm bảo rằng tài nguyên được giải phóng.");
        }
    }
}

Kết quả:

Lỗi: nonexistentfile.txt (No such file or directory)
Đảm bảo rằng tài nguyên được giải phóng.

4. Các Lớp Liên Quan Đến I/O Khác Link to heading

  • ObjectInputStream và ObjectOutputStream: Dùng để đọc và ghi các đối tượng Java.
  • DataInputStream và DataOutputStream: Dùng để đọc và ghi dữ liệu kiểu nguyên thủy (int, float, double, v.v.).
  • FileChannel: Là một phần của NIO (New I/O), cung cấp các phương thức hiệu quả để thao tác với tệp tin.

5. Kết luận Link to heading

Java I/O cung cấp một hệ thống linh hoạt và mạnh mẽ để làm việc với dữ liệu. Bạn có thể xử lý các tệp tin nhị phân, văn bản, và thậm chí là các đối tượng Java. Việc sử dụng đúng các lớp và giao diện trong Java I/O sẽ giúp bạn tối ưu hóa hiệu suất và độ chính xác của ứng dụng. Quay lại Chuyển sang bài học tiếp theo
