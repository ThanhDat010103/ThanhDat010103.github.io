Lập Trình Mạng (Networking) trong Java Link to heading

Lập trình mạng trong Java cho phép các ứng dụng giao tiếp với nhau qua các giao thức mạng như TCP/IP hoặc UDP. Java cung cấp một bộ API mạnh mẽ trong gói java.net để tạo các ứng dụng mạng, giúp bạn dễ dàng thiết lập các kết nối mạng, gửi và nhận dữ liệu qua mạng.

1. Các Giao Thức Mạng Chính Link to heading

Java hỗ trợ nhiều giao thức mạng phổ biến:

  • TCP (Transmission Control Protocol): Giao thức kết nối đáng tin cậy, đảm bảo dữ liệu được truyền tải đầy đủ và chính xác.
  • UDP (User Datagram Protocol): Giao thức không kết nối, nhẹ nhàng hơn nhưng không đảm bảo độ tin cậy.

2. Cấu Trúc của Lập Trình Mạng Link to heading

Lập trình mạng trong Java chủ yếu sử dụng các đối tượng sau:

  • Socket: Dùng để tạo kết nối TCP.
  • ServerSocket: Dùng để tạo một server lắng nghe các kết nối từ client.
  • DatagramSocket: Dùng để gửi và nhận dữ liệu qua UDP.
  • DatagramPacket: Dùng để chứa dữ liệu trong giao thức UDP.

3. Kết Nối TCP trong Java Link to heading

3.1. Máy Chủ TCP (Server) Link to heading

Để tạo một server TCP, chúng ta sử dụng ServerSocket. Server sẽ lắng nghe các yêu cầu kết nối từ các client. Khi nhận được kết nối, server tạo ra một socket riêng biệt để giao tiếp với client.

Ví dụ về Máy Chủ TCP: Link to heading 

import java.io.*;
import java.net.*;

public class TCPServer {
    public static void main(String[] args) {
        try {
            // Tạo ServerSocket và lắng nghe kết nối
            ServerSocket serverSocket = new ServerSocket(1234);
            System.out.println("Server đang lắng nghe tại cổng 1234...");

            // Chấp nhận kết nối từ client
            Socket clientSocket = serverSocket.accept();
            System.out.println("Client kết nối từ: " + clientSocket.getInetAddress());

            // Tạo các luồng để đọc và ghi dữ liệu
            BufferedReader reader = new BufferedReader(new InputStreamReader(clientSocket.getInputStream()));
            PrintWriter writer = new PrintWriter(clientSocket.getOutputStream(), true);

            // Đọc dữ liệu từ client và phản hồi
            String message = reader.readLine();
            System.out.println("Nhận từ client: " + message);
            writer.println("Chào client! Bạn đã gửi: " + message);

            // Đóng kết nối
            reader.close();
            writer.close();
            clientSocket.close();
            serverSocket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

3.2. Máy Khách TCP (Client) Link to heading

Máy khách sử dụng Socket để kết nối đến server và gửi/nhận dữ liệu.

Ví dụ về Máy Khách TCP: Link to heading 

import java.io.*;
import java.net.*;

public class TCPClient {
    public static void main(String[] args) {
        try {
            // Kết nối đến server tại địa chỉ và cổng
            Socket socket = new Socket("localhost", 1234);
            System.out.println("Kết nối tới server...");

            // Tạo các luồng để gửi và nhận dữ liệu
            BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
            PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);

            // Gửi thông điệp đến server
            System.out.print("Nhập thông điệp gửi đến server: ");
            String message = reader.readLine();
            writer.println(message);

            // Đọc phản hồi từ server
            BufferedReader serverReader = new BufferedReader(new InputStreamReader(socket.getInputStream()));
            String response = serverReader.readLine();
            System.out.println("Nhận từ server: " + response);

            // Đóng kết nối
            reader.close();
            writer.close();
            serverReader.close();
            socket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Kết quả:

  • Server nhận và phản hồi thông điệp của client.
  • Client gửi thông điệp đến server và nhận lại phản hồi.

4. Kết Nối UDP trong Java Link to heading

Giao thức UDP không yêu cầu kết nối trực tiếp và thích hợp cho các ứng dụng yêu cầu tốc độ truyền nhanh, nhưng không cần độ tin cậy như trong truyền tải video, âm thanh.

4.1. Máy Chủ UDP Link to heading

Máy chủ UDP sử dụng DatagramSocket để nhận dữ liệu từ các client và gửi lại phản hồi.

Ví dụ về Máy Chủ UDP: Link to heading 

import java.net.*;

public class UDPServer {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket(1234);
            System.out.println("Server UDP đang lắng nghe tại cổng 1234...");

            byte[] receiveData = new byte[1024];

            // Nhận dữ liệu từ client
            DatagramPacket receivePacket = new DatagramPacket(receiveData, receiveData.length);
            socket.receive(receivePacket);
            String message = new String(receivePacket.getData(), 0, receivePacket.getLength());
            System.out.println("Nhận từ client: " + message);

            // Gửi phản hồi đến client
            InetAddress clientAddress = receivePacket.getAddress();
            int clientPort = receivePacket.getPort();
            String response = "Chào client! Bạn đã gửi: " + message;
            DatagramPacket sendPacket = new DatagramPacket(response.getBytes(), response.length(), clientAddress, clientPort);
            socket.send(sendPacket);

            socket.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

4.2. Máy Khách UDP Link to heading

Máy khách UDP sử dụng DatagramSocket để gửi dữ liệu đến máy chủ và nhận phản hồi.

Ví dụ về Máy Khách UDP: Link to heading 

import java.net.*;

public class UDPClient {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket();
            InetAddress serverAddress = InetAddress.getByName("localhost");

            // Gửi thông điệp đến server
            String message = "Chào server UDP!";
            DatagramPacket sendPacket = new DatagramPacket(message.getBytes(), message.length(), serverAddress, 1234);
            socket.send(sendPacket);

            // Nhận phản hồi từ server
            byte[] receiveData = new byte[1024];
            DatagramPacket receivePacket = new DatagramPacket(receiveData, receiveData.length);
            socket.receive(receivePacket);
            String response = new String(receivePacket.getData(), 0, receivePacket.getLength());
            System.out.println("Nhận từ server: " + response);

            socket.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

Kết quả:

  • Server nhận và phản hồi thông điệp của client.
  • Client gửi thông điệp đến server và nhận lại phản hồi.

5. Các Lớp Liên Quan Đến Networking Link to heading

  • Socket: Dùng để thiết lập kết nối TCP giữa client và server.
  • ServerSocket: Dùng để tạo một server TCP và lắng nghe các kết nối đến.
  • DatagramSocket: Dùng để gửi và nhận dữ liệu qua UDP.
  • DatagramPacket: Dùng để chứa dữ liệu trong giao thức UDP.
  • URL và URLConnection: Dùng để làm việc với URL và các kết nối HTTP.
