Java Database Connectivity (JDBC) Link to heading

Java Database Connectivity (JDBC) là một API trong Java giúp các ứng dụng Java kết nối và thao tác với cơ sở dữ liệu. JDBC cung cấp các phương thức để truy vấn, cập nhật, và thao tác với dữ liệu trong cơ sở dữ liệu như MySQL, PostgreSQL, Oracle, và nhiều hệ quản trị cơ sở dữ liệu khác.

JDBC sử dụng các lớp và giao diện để thiết lập kết nối với cơ sở dữ liệu, gửi các câu lệnh SQL và nhận kết quả trả về.

1. Cấu Trúc của JDBC Link to heading

JDBC có thể được chia thành ba thành phần chính:

  1. JDBC Drivers: Là các lớp cung cấp giao diện giữa Java và các cơ sở dữ liệu.
  2. Connection: Đối tượng dùng để thiết lập kết nối tới cơ sở dữ liệu.
  3. Statement: Đối tượng dùng để thực thi các câu lệnh SQL.
  4. ResultSet: Đối tượng chứa kết quả truy vấn từ cơ sở dữ liệu.

2. Cách Kết Nối Với Cơ Sở Dữ Liệu Link to heading

Để kết nối với cơ sở dữ liệu, bạn cần thực hiện các bước sau:

  1. Tải JDBC Driver: Tải JDBC driver phù hợp với hệ quản trị cơ sở dữ liệu mà bạn đang sử dụng.
  2. Kết nối đến cơ sở dữ liệu: Dùng đối tượng Connection để kết nối.
  3. Tạo đối tượng Statement: Để gửi câu lệnh SQL đến cơ sở dữ liệu.
  4. Xử lý kết quả: Dùng ResultSet để lưu trữ và xử lý kết quả trả về từ cơ sở dữ liệu.

2.1. Cách Kết Nối JDBC với MySQL Link to heading

Đầu tiên, bạn cần thêm JDBC driver của MySQL vào dự án của bạn. Nếu sử dụng Maven, bạn có thể thêm dependency sau vào file pom.xml:

<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <version>8.0.23</version>
</dependency>

2.2. Ví Dụ về Kết Nối Cơ Bản và Truy Vấn Dữ Liệu Link to heading

Bước 1: Kết nối với cơ sở dữ liệu Link to heading 

import java.sql.*;

public class JDBCExample {
    public static void main(String[] args) {
        // URL kết nối tới cơ sở dữ liệu MySQL
        String url = "jdbc:mysql://localhost:3306/mydatabase";
        String username = "root";
        String password = "password";

        try {
            // Tạo kết nối đến cơ sở dữ liệu
            Connection connection = DriverManager.getConnection(url, username, password);
            System.out.println("Kết nối thành công!");

            // Tạo statement để thực thi câu lệnh SQL
            Statement statement = connection.createStatement();

            // Truy vấn dữ liệu từ bảng "employees"
            String sql = "SELECT * FROM employees";
            ResultSet resultSet = statement.executeQuery(sql);

            // Xử lý kết quả trả về
            while (resultSet.next()) {
                int id = resultSet.getInt("id");
                String name = resultSet.getString("name");
                double salary = resultSet.getDouble("salary");
                System.out.println("ID: " + id + ", Tên: " + name + ", Lương: " + salary);
            }

            // Đóng kết nối
            resultSet.close();
            statement.close();
            connection.close();

        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

Kết quả (Giả sử cơ sở dữ liệu có bảng “employees”):

Kết nối thành công!
ID: 1, Tên: John, Lương: 50000.0
ID: 2, Tên: Jane, Lương: 60000.0

3. Các Loại Statement trong JDBC Link to heading

JDBC cung cấp ba loại Statement để thực thi các câu lệnh SQL:

3.1. Statement Link to heading

Dùng cho các câu lệnh SQL đơn giản như SELECT, INSERT, UPDATE, DELETE.

Ví dụ: Link to heading 

Statement statement = connection.createStatement();
String sql = "SELECT * FROM employees";
ResultSet resultSet = statement.executeQuery(sql);

3.2. PreparedStatement Link to heading

Dùng khi bạn cần thực thi các câu lệnh SQL có tham số động, giúp bảo mật tốt hơn và tối ưu hiệu suất khi thực thi nhiều lần.

Ví dụ: Link to heading 

String sql = "INSERT INTO employees (name, salary) VALUES (?, ?)";
PreparedStatement preparedStatement = connection.prepareStatement(sql);
preparedStatement.setString(1, "Alice");
preparedStatement.setDouble(2, 70000.0);
preparedStatement.executeUpdate();

3.3. CallableStatement Link to heading

Dùng để gọi các thủ tục (stored procedures) trong cơ sở dữ liệu.

Ví dụ: Link to heading 

CallableStatement callableStatement = connection.prepareCall("{call getEmployeeById(?)}");
callableStatement.setInt(1, 1);
ResultSet resultSet = callableStatement.executeQuery();

4. Xử Lý Kết Quả với ResultSet Link to heading

Sau khi thực thi câu lệnh SELECT, kết quả trả về sẽ được lưu trữ trong một đối tượng ResultSet. Bạn có thể sử dụng các phương thức của ResultSet để truy xuất dữ liệu.

4.1. Các Phương Thức Thường Dùng của ResultSet Link to heading

  • getInt(String columnName): Lấy giá trị kiểu int của cột.
  • getString(String columnName): Lấy giá trị kiểu String của cột.
  • getDouble(String columnName): Lấy giá trị kiểu double của cột.
  • next(): Di chuyển con trỏ đến dòng kết quả tiếp theo.

4.2. Ví Dụ Sử Dụng ResultSet Link to heading 

while (resultSet.next()) {
    int id = resultSet.getInt("id");
    String name = resultSet.getString("name");
    double salary = resultSet.getDouble("salary");
    System.out.println("ID: " + id + ", Tên: " + name + ", Lương: " + salary);
}

5. Cập Nhật Dữ Liệu với JDBC Link to heading

JDBC cũng hỗ trợ các câu lệnh SQL như INSERT, UPDATE, và DELETE để thay đổi dữ liệu trong cơ sở dữ liệu.

5.1. Ví Dụ về Cập Nhật Dữ Liệu Link to heading 

String sql = "UPDATE employees SET salary = ? WHERE id = ?";
PreparedStatement preparedStatement = connection.prepareStatement(sql);
preparedStatement.setDouble(1, 80000.0);
preparedStatement.setInt(2, 1);
preparedStatement.executeUpdate();

5.2. Ví Dụ về Xóa Dữ Liệu Link to heading 

String sql = "DELETE FROM employees WHERE id = ?";
PreparedStatement preparedStatement = connection.prepareStatement(sql);
preparedStatement.setInt(1, 2);
preparedStatement.executeUpdate();

6. Xử Lý Ngoại Lệ trong JDBC Link to heading

JDBC sử dụng ngoại lệ SQLException để báo cáo lỗi khi thực hiện các thao tác với cơ sở dữ liệu. Bạn cần xử lý các ngoại lệ này để đảm bảo ứng dụng của bạn không bị sập khi gặp lỗi kết nối hoặc lỗi truy vấn.

Ví Dụ về Xử Lý Ngoại Lệ Link to heading 

try {
    // Kết nối và thực hiện các câu lệnh SQL
} catch (SQLException e) {
    System.out.println("Lỗi SQL: " + e.getMessage());
} finally {
    // Đảm bảo đóng kết nối và các tài nguyên
}

7. Kết Luận Link to heading

Java Database Connectivity (JDBC) là một API quan trọng giúp các ứng dụng Java giao tiếp với cơ sở dữ liệu. Với JDBC, bạn có thể thực hiện các thao tác như kết nối, truy vấn, cập nhật, và xóa dữ liệu từ cơ sở dữ liệu. Hiểu rõ các khái niệm như Connection, Statement, PreparedStatement, và ResultSet sẽ giúp bạn phát triển các ứng dụng Java hiệu quả và mạnh mẽ. Quay lại Chuyển sang bài học tiếp theo
